## Key Characters

`` Power
`⌘` Command ( Cmd )
`⌥` Option | Alt ( on Windows )
`^` Control ( Ctrl )
`⇧` Shift ( Shf )
`⇪` CapsLock
`⇥` Tab
`⎋` Escape ( Esc )
`↩` Return ( Ret ) | Enter ( on Windows )
`凵` Space

`↑` Up
`↓` Down ( Dn )
`←` Left ( Lf )
`→` Right ( Rg )

`⌫` Delete ( on Mac ) | Backspace ( on Windows )
`Num` Digits: 0 ~ 9

## Abbreviations

- Button

    `Cmd` Command
    `Ctrl` Control
    `Esc` Escape
    `Opt` Option
    `Ret` Return
    `Shf` Shift
    `Del` Delete

- Direction & Position

    `Dn` Down
    `Lf` left
    `Rg` Right

    `Prev` Previous
    `Btm` Bottom

- Word

    `App` Application
    `Btn` Button
    `Dir` Directory

    `Mv` Move
    `Rm` Remove

    `Ld` Leader
    `Pf` Prefix

- Symbol

    `&` And
    `|` Or

## Examples

- `⌘ ⌥ a` Manipulation

    Press "Cmd", "Opt" & "a" at the same time to do the manipulation .

- `A` Manipulation

    The capital letter "A" means pressing "Shf" & "a" at the same time !

- `Shortcut1 | Shortcut2` Manipulation

    The manipulation has two shortcuts Shortcut1 & Shortcut2 .

- `⌘ [ 1 | 2 ]` Manipulation

    Press "⌘ 1" or "⌘ 2" to do the manipulation .
