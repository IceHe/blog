子问题：
PHP 空字符串""、布尔值FALSE、空NULL 的区别

通过编程求证，
以下是显示的结果：
param=array(3) {
  [0]=>
  string(0) ""
  [1]=>
  bool(false)
  [2]=>
  NULL
}
Remember that param[3] does not exist!


empty 和 isset 的区别
empty("")=bool(true)
empty(FALSE)=bool(true)
empty(NULL)=bool(true)
empty(undefined)=bool(true)

isset("")=bool(true)
isset(FALSE)=bool(true)
isset(NULL)=bool(false)
isset(undefined)=bool(false)

以上运行结果，带有PHP的错误报告，是对不存在参数的提醒。
调用不存在的参数实际得到是NULL

接下来关闭错误报告对于提醒Notice报告的显示。


空字符串""，布尔否FALSE，空NULL 之间的对比
全等 ===
""===FALSE :bool(false)
""===NULL :bool(false)
FALSE===NULL :bool(false)

等于 ==
""==FALSE :bool(true)
""==NULL :bool(true)
FALSE==NULL :bool(true)

不全等 !==
""!==FALSE :bool(true)
""!==NULL :bool(true)
FALSE!==NULL :bool(true)

不等于 !=
""!=FALSE :bool(false)
""!=NULL :bool(false)
FALSE!=NULL :bool(false)


!== 含义的探索
TRUE!==FALSE :bool(true)
"abc"!=="xyz" :bool(true)
由以上结果可知!==含义为"不全等"，
而非"全不等！"

string!==BOOLEAN :
"abc"!==TRUE :bool(true)
"xyz"!==FALSE :bool(true)

others!==NULL :
"abc"!==NULL :bool(true)
FALSE!==NULL :bool(true)


不存在的参数 与 ""、FALSE、NULL 对比
全等 ===
(undefined)==="" :bool(false)
(undefined)===FALSE :bool(false)
(undefined)===NULL :bool(true)

等于 ==
(undefined)=="" :bool(true)
(undefined)==FALSE :bool(true)
(undefined)==NULL :bool(true)

不全等 !==
(undefined)!=="" :bool(true)
(undefined)!==FALSE :bool(true)
(undefined)!==NULL :bool(false)

不等于 !=
(undefined)!="" :bool(false)
(undefined)!=FALSE :bool(false)
(undefined)!=NULL :bool(false)
