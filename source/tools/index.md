## **Apps**

*以下为Mac平台下的软件。*

### **Common**

- **Evernote (Premium)**

	第二大脑，知识管理工具。

	*好记性不如烂笔头。
	还是书写记得更牢，但已是电子时代，
	知识更新之快，纸笔难跟上。*

	*用过很多同类软件，还是买了它的Premium版。*
	
- **Doit.im (Pro)**

	事务管理、日程规划。 *有更好的替代App，只是用习惯了。*

- **Shawdowsocks**

	首推的翻G_F_W工具。

- **Firefox**

	配合VimFX拓展，可在浏览器内全局使用Vim键位浏览网页！键盘党神器。

	*Vimperator拓展功能太强大，响应、外观和易用性不如VimFX。
	虽然Chrome很好，但更耗电，影响到了Mac的续航；
	而且Vimium等拓展无法在页面未加载完毕的情况下使用Vim键位，不好用。*

- **Chrome**

	Web的调试更习惯Chrome的Inspect Element，而非Firebug。
<br/>

### **Develop**

- **PyCharm, &nbsp;PhpStorm**

	优秀的IDE，用于Python、PHP，配合Vim插件用Vim键位编码。 
	
	*JetBrains出品，必属精品。还有IntelliJ IDEA、WebStorm、RubyMine等。*

- **iTerm2**

	强大易用的命令行工具。*系统默认Terminal的替代品。*

- **Parallel Desktop**

	极为优秀的虚拟机软件，
	将Windows、Ubuntu等操作系统的使用无缝贴合到 OS X 中去。

- **Squel Pro**

	管理MySQL数据库的客户端工具，方便好用。

- **Transmit**

	FTP客户端工具，方便好用。

- **Charles**

	网络封包分析工具。*Windows的Fiddler替代品。*

- **Dash**

	程序员API参考文档浏览器，代码片段管理工具，功能单一却精准。

- **Sublime Text**

	简单易用的代码编辑器，我只用于编辑一般的文本。
<br/>

### **System Tools**

- **EuDic Free**

	欧路词典，Mac上最好用的词典，免费版就够用了。

- **Manico**

	App的快速启动/切换工具。

	*Windows上已习惯迅速用快捷键与手势打开/切换程序，
	Mac的触摸板极好用，但我仍是不想老动光标的键盘党。*

- **Moom**

	窗口布局软件，便捷地移动、缩放窗口，让你更好地利用桌面空间。

- **HyperDock**

	窗口增强工具。光标停到Dock的软件图标上，能快速预览该软件的所有窗口，
	点击切换不同到不同的窗口或桌面，还有快速调整窗口布局、大小、位置等的功能。

- **Owly**

	防止Mac休眠的软件，顺带小巧的倒计时功能。

	*因为Mac续航再好，偶尔的休眠还是会关闭一些软件，中断工作。
	它比Caffine好看、好用。*

- **ClipMenu**

	剪贴板管理工具，支持纯文本、图片、RTF、PDF、URL等各种格式，
	包括剪贴历史记录、排序、大小写转换、自定义文段的调出。

- **CheatSheet**

	快捷键快速提醒工具。长按Command键，显示当前程序的快捷键列表。
	Mac快捷键入门神器。

- **iStat Menus**

	硬件信息检测软件，我看重其支持系统顶栏部分自定义的功能。
	方便查看CPU、内存、网络、硬盘及其它硬件的状态，还有日期时间栏的定制。

- **CleanMyMac**

	Mac系统清理软件。

- **MPlayerX**

	优秀的视频播放软件。

- **Boom**

	音效增强软件。

- **Nozio**

	白噪音应用。模拟大自然的声音，屏蔽其它噪音干扰，
	帮助你放松睡眠，或保持专注、提高效率。

- **Time Out**

	定时作息提醒工具。定时覆盖屏幕，提醒你离开电脑，去休息眼睛和身体。

- **f.lux**

	根据所在地当时的日照情况，自动调整屏幕色温、亮度，以保护眼睛。

- **LastPass**

	类似与1Password的密码管理工具。

- **One 一览**

	一款小巧的聚合型资讯订阅软件。

- **Gas Mask**

	便捷的Hots文件修改工具。

<br/><i>不使用 **Alfred**，
因为我连用Spotlight的机会都不多，
要用命令的话，也会直接使用命令行工具，
没有Alfred之类软件的用武之地。

也不使用额外的 **PDF阅读软件**，
Mac系统自带的Preview和Firefox就够用了。</i>
<br/>

### **iOS**

- **随手记专业版**

	首推的记账软件。

- **多看阅读**

	阅读软件。可导入PDF，带云存储，阅读、操作体验一流，

- **扇贝系列App**

	扇贝单词、扇贝炼句、扇贝听力、扇贝读书。

	*工具不重要，最重要还是要坚持学。*

- **网易公开课**

- **Douban FM**

	*豆瓣FM，随便听听、或者想要发现喜欢的歌的时候，还是会选择它。Pro版已入手。*

- **喜马拉雅FM、凤凰FM、企鹅FM**

	*喜马拉雅FM，使用体验与内容俱佳，许多电台自媒体的首选独播平台。
	凤凰FM，拥有凤凰传媒的版权内容，有我最喜欢谈话节目《锵锵三人行》。
	企鹅FM，腾讯出品用户体验必然好，娱乐类节目偏多，不适合我，
	但还是因为有我喜欢栏目在，只能留下。*

	*基本所有电台类App我都用过，最后剩下它们。
	因为内容为王，我通常都为了听喜欢的自媒体栏目被迫换FM类App的，
	然后发现产品体验好，就自然而然留下来了。*

- **Netease Music、QQ Music**

	*网易云音乐，音乐软件的新标杆，用户体验感觉被QQ音乐追上来了，
	但是找好歌单、冷门歌曲，在一首歌下面评论分享感受，还是比QQ音乐的弹幕模式靠谱…
	QQ音乐，有很多独家版权没办法。*

- **Bilibili、AcFun**

	*现在看视频不带弹幕，就觉得心缺了一块，食之无味。*
